FunPay Parsers
==============

FunPay Parsers - это набор парсеров для популярной торговой площадки `FunPay <https://funpay.com>`_.
Они позволят вам без лишних проблем преобразовать полученный HTML/JSON в удобные и структурированные датаклассы для
дальнейшего использования в вашей программе.

.. grid:: 1 2 2 3

    .. grid-item::

        .. card::

            :fas:`bolt;pst-color-primary` Скорость
            ^^^
            FunPay Parsers использует `Selectolax <https://github.com/rushter/selectolax>`_ на `Lexbor <https://github.com/lexbor/lexbor>`_,
            что делает скорость парсинга крайне высокой. Например, парсинг 4000 лотов занимает ~0.2с!

    .. grid-item::

        .. card::

            :fas:`percent;pst-color-primary` Покрытие
            ^^^
            С помощью FunPay Parser можно спарсить 99% всех сущностей FunPay. Начиная с бейджиков и заканчивая целыми страницами.

    .. grid-item::

        .. card::

            :fas:`code;pst-color-primary` Удобство
            ^^^
            Парсеры преобразуют HTML в удобные и структурированные датаклассы.
